# Twitter Auto DMs

Send automated messages (DMs) to your new Twitter followers (work-in-progress)

![Imgur Image](http://i.imgur.com/71UgxiP.jpg)

## Installation

```bash
$ npm install
```

## Running the app

Copy `.env.sample` to `.env` and add your environment variables.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
